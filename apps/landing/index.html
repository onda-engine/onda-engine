What's inside

A renderer-first engine, end to end

The renderer is the product; React, JSON and AI are all adapters onto one universal scene graph.

  • GPU-native Vello renderer

    Anti-aliased fills and strokes, real rounded rects, arbitrary Bézier paths, linear/radial gradients, clip masks, and native per-glyph vector text.

  • Author in React / JSX

    <Composition>, <Rect>, <Path>, <Sequence>, useCurrentFrame, interpolate, spring. The Remotion DX you know, compiled to a scene graph.

  • Universal scene graph

    React, JSON or an AI agent all emit the same graph. One runtime renders it — the renderer is the platform, not any single front end.

  • SVG import

    <Svg src=… | markup=…> expands logos and icons into native vector nodes — animate them like any other primitive.

  • CPU + GPU backends

    --backend auto|vello|cpu. A CPU reference rasterizer produces bit-identical, deterministic output for CI and golden-image tests.

  • Export PNG / GIF / MP4

    Drive everything from the onda CLI: render, export, export-frames. Scale across cores and GPU workers.
