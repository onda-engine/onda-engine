# @onda-engine/wasm-audio

ONDA **audio analysis** (decode + FFT spectrum + beat / onset / tempo detection) compiled to WebAssembly — drives audio-reactive visuals in the browser, producing **identical spectra to the native export** so preview matches the final file.

> **Low-level.** Usually consumed via [`@onda-engine/components`](https://www.npmjs.com/package/@onda-engine/components) (e.g. `useAudioBeats`) and the player.

[Docs](https://onda.video) · [GitHub](https://github.com/onda-engine/onda-engine) · [Onda Studio](https://studio.onda.video)

## Install

```bash
npm install @onda-engine/wasm-audio
```
