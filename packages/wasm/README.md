# @onda-engine/wasm

The ONDA renderer compiled to **WebAssembly** — the real native engine in the browser (no DOM, no Chromium). Powers author-time **text metrics** and the CPU raster path behind ONDA's live-preview pipeline.

> **Low-level.** You usually don't install this directly — it ships as a dependency of [`@onda-engine/player`](https://www.npmjs.com/package/@onda-engine/player) and [`@onda-engine/components`](https://www.npmjs.com/package/@onda-engine/components).

[Docs](https://onda.video) · [GitHub](https://github.com/onda-engine/onda-engine) · [Onda Studio](https://studio.onda.video)

## Install

```bash
npm install @onda-engine/wasm
```
