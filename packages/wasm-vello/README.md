# @onda-engine/wasm-vello

The ONDA **Vello (GPU) renderer** compiled to WebAssembly — [Vello](https://github.com/linebender/vello) in the browser over **WebGPU** (no DOM, no Chromium). This is the GPU preview path behind `@onda-engine/player`; it draws the same scene graph the native renderer exports.

> **Low-level / preview path.** Usually consumed via [`@onda-engine/player`](https://www.npmjs.com/package/@onda-engine/player). WebGPU support varies by browser.

[Docs](https://onda.video) · [GitHub](https://github.com/onda-engine/onda-engine) · [Onda Studio](https://studio.onda.video)

## Install

```bash
npm install @onda-engine/wasm-vello
```
